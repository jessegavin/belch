# belch

> Play a sound. Useful for many things.

## Install

```bash
npm install -g belch
```

## Usage

default:

```bash
belch
```

other options:

```bash
belch --help
```

## OS Notes

Supports Windows, MacOS and Linux. For Linux, you must first install `sox` for playing mp3s. E.g. on Ubuntu do the following:

```bash
sudo apt-get install sox libsox-fmt-mp3
```
